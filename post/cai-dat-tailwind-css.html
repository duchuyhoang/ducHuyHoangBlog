Tìm hiểu về Tailwind CSS và cài đặt cho dự án React

09-10-2022

-

10 minutes read

-
Bình luận
Đức Huy Hoàng

Bạn đang lo lắng vì đặt tên class, sợ bị trùng css không mong muốn,hay đang muốn một framework giống bootstrap để tận dụng được các class dùng chung mà không viết lại quá nhiều. TailwindCSS là framework dành cho bạn

Tailwind CSS là gì ?

  • Tailwind css là một utility-first CSS framework nó hỗ trợ phát triển xây dựng nhanh chóng giao diện người dùng, nó cũng có điểm chung giống như Bootstrap & điểm làm nó nổi bật hơn cả đó là chúng ta có thể tùy biến phát triển css theo cách mà chúng ta định nghĩa ra.
  • Hoặc là chúng ta có thể thêm các class của tailwind vào 1 class để dùng chung sau này

Cài đặt Tailwind CSS cho dự án React

Sau đây mình sẽ giới thiệu cho các bạn cách cài đặt Tailwind trong Reactjs nhé. Chúng ta dùng Create React App nhé

  • Đầu tiên các bạn tạo 1 dự án React:

newReactApp.sh

// Init project của chúng ta
npx create-react-app react-with-tailwind
cd react-with-tailwind
  • Tiếp theo chúng ta cài các dependency cần thiết:

installDependency.sh

npm install -D tailwindcss postcss autoprefixer
// Sau đó init tailwind CSS trong project của chúng ta
npx tailwindcss init -p
  • Okay chúng ta vào file tailwind.config.js vừa tạo và config như sau:

tailwind.config.js

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ['./src/**/*.{html,js,jsx,ts,tsx}'],
  theme: {
    extend: {}
  },
  plugins: []
}
  • Và thêm @tailwind directives vào file index.css của bạn:

index.css

@tailwind base;
@tailwind components;
@tailwind utilities;

Chú ý

Kiểm tra package.json của bạn nếu react-scripts >= 5 thì có thể chạy ok
Còn nếu package.json của bạn nếu react-scripts < 5 thì bạn cần phải config thêm craco: Ở đây

  • Cuối cùng là thử chạy nào:

App.js

1export default function App() {
2  return <h1 className="text-3xl font-bold underline">Hello world!</h1>
3}

Tổng kết
