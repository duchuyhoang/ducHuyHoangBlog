Các extension xịn xò mà mình dùng trong Visual Studio Code

13-10-2022
-

5 minutes read

-
Bình luận
Đức Huy Hoàng

Bạn là một developer và bạn dùng Visual Studio Code. Visual Studio Code là 1 text editor khá mới nhưng free, chạy được trên nhiều hệ điều hành. Tuy chỉ là text editor nhưng ta vẫn có thể compile, run, debug code trên VS Code, không thua gì cái IDE xịn. Vừa nhẹ, vừa free, lại mạnh mẽ, nên dân web developer như mình rất khoái dùng VS Code. Sau đây mình sẽ giới thiệu cho các bạn những extension xịn xò nhen

Danh sách extension mình hay dùng

  • GitLens: VSCode đã hỗ trợ pull/push từ Git, nhưng GitLens bá đạo hơn nhiều. Nó giúp bạn biết từng dòng code do ai viết, viết vào lúc nào, nằm trong commit nào. Bạn cũng có thể tua ngược về quá khứ để xem file đã thay đổi như thế nào, rất tiện

  • Color Highlight: Color Highlight hỗ trợ hightlight màu ở editor của bạn.

  • vscode-icons: Extension icon cho file trong VS code của bạn.

  • Import Cost: Extension cho phép chúng ta biết được thư viện nào quá nặng, nên thay thế hoặc loại bỏ, hoặc import 1 phần nhỏ thôi

  • Code Spell Checker: Extension này sẽ highlight những đoạn sai chính tả để mình sửa nha.

  • Path Intellisense: Path Intellisense giúp chúng ta trong giai đoạn import các module(có thể show tên các file thậm chí cả hidden file nếu được config)

  • Cuối cùng là cặp đôi ESLint và Prettier

Extension dành cho web developer

Bên trên là những extension nên dùng dành cho mọi dev, nhưng mà còn web dev như mình thì sao. Mình xin giới thiệu các bạn những extension hay ho cho dân làm web

  • HTML CSS Support:Làm việc với web chắc chắn không thể không biết HTML,CSS rồi. Một extension hữu ích cho web developer.

  • Live Server: Một extension rất tiện để chúng ta phát triển HTML, CSS mà k cần reload lại page.

  • JavaScript (ES6): Một extension hữu ích dành cho Javascript dev. Extension này bao gồm các snippets cho JavaScript trong cú pháp của ES6.

  • npm Intellisense: Giúp cho việc tự động điền npm modules trong câu lệnh import.

  • YAML: Extension hỗ trợ cho YAML file.

Nên cài extension tùy vào mục đích của bạn !!!

Ngoài ra, tuỳ vào ngôn ngữ/framework đang dùng mà các bạn có thể cài thêm Code Snippets hoặc VSCode Extension cho ngôn ngữ đó. Bản thân mình code React, Vue nên cài thêm rất nhìu luôn á.

Tổng kết

Trong bài này , mình đã giới thiệu cho các bạn những extension bá đạo mà các bạn nên dùng rồi. Cài đủ các extension các bạn sẽ có 1 bản VS Code khá mạnh, đủ xài để code web nhé.
